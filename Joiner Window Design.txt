Joiner Window Design

For Wing Panel 1 only add another parameter window tab called Joiner.  The parameters are:

"Add Rib 1a" checkbox which puts an extra rib between ribs 1 & 2.  If selected, this rib must be DXF exportable.
Label that says "Dihedral Joiner(s)"
If the spars are Top or Bottom Spar then display a checkbox for "Center Spar Wood Joiner"
Display a checkbox for "Joiner behind mid spar".  If checked display CF Rod, CF Tube, and Aluminum Tube mutually exclusive options.
	Below that display OD/ID parameters options as appropriate base on tube/rod selection above.
Display a checkbox for "Joiner at 50% Chord".  If checked display CF Rod, CF Tube, and Aluminum Tube mutually exclusive options.
	Below that display OD/ID parameters options as appropriate base on tube/rod selection above.

The "Center Spar Wood Joiner" extends like a shear web between the spars, but it cuts through rib 1 and rib 1a (if present). It is displayed up to, but doesn't cut rib 2.
The other Joiners should be extruded/cut through rib 1, 1a and the outer edge of rib 2.

Add defaults for all of these and Edit/Defaults should allow you to change them persistently.
