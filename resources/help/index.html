Basic workflow

  1. Choose the number of wing panels and enter each panel's parameters.
  2. Import root and tip airfoil .dat files as needed.
  3. Select the spars, sheeting, leading/trailing edges, controls, and joiners.
  4. Press Update View. Parameter edits alone do not rebuild geometry.
  5. Inspect the complete mirrored wing in 3D View.
  6. Press Generate Plan to create the flattened full-scale drawing.
  7. Save the project, export the plan to PDF, or export selected cutting parts to DXF/SVG.
Wing convention: parameters describe one half-wing. DesignRC constructs that half from the center outward and mirrors it for the complete 3D preview and plan.

Global Units and Global/mm/in

Global Units

The global unit is the project's normal display and entry unit. It can be set to millimeters or inches in Edit > Defaults. The setting determines how length controls marked Global, calculated metrics, and most plan annotations are shown. Internally, geometry remains dimensionally equivalent when the display unit changes.

The Global, mm, and in selector

Where unit overrides are available, the selector beside a length has three meanings:

Global
Use the current Global Units setting. Changing Global Units changes this field's display unit.
mm
Always display and edit this particular value in millimeters, regardless of Global Units.
in
Always display and edit this particular value in inches, regardless of Global Units.

Unit overrides belong to the parameter, so plan annotations derived from that parameter use the same unit.

Entering inch measurements

  • Spinner arrows move by exactly 1/32 inch and have no application-imposed upper limit.
  • Values exactly on the 1/32-inch grid display as reduced fractions, such as 5/32 in or 1 3/4 in.
  • A manually entered decimal is retained as a decimal unless it equals a multiple of 1/32 inch.
  • Fractional input is accepted in forms such as 3/16 and 1 1/2.

Wing panels

Number of Wing Panels
Sets how many successive panels make up one half-wing. Increasing or decreasing it adds or removes parameter tabs after confirmation.
Panel 1
The center panel. Its root is at the aircraft centerline and it alone can use Rib 1a.
Panels 2-N
Outer panels. Each panel automatically inherits the preceding panel's tip chord and tip airfoil as its root chord and root airfoil, so no separate root-chord option is shown.

Panel geometry is flattened for plans but assembled with dihedral in 3D. Joint ribs bisect the angle between adjacent panels so their faces meet. Intermediate ribs are perpendicular to that panel's leading/trailing-edge direction.

Specs tab

Airfoils

Import root .dat
Loads a normalized airfoil coordinate file for the panel root. For outer panels the root airfoil comes from the preceding panel.
Import tip .dat
Loads the airfoil at the panel tip. Ribs between root and tip interpolate between the two profiles.

Dimensions

Panel Span
The root-to-tip length of this flattened half-wing panel.
Root Chord
Root airfoil chord. It is editable only on Panel 1; outer panels inherit it.
Tip Chord
Chord at the panel's outer rib.
Tip Sweep
Fore/aft offset of the tip leading edge relative to the root leading edge.
Dihedral
The angular change associated with this panel. DesignRC assembles adjoining panels and bisects their joint angle at the shared ribs.
Tip Twist
Wash-in/wash-out applied progressively from zero at the root to the entered angle at the tip. Positive values rotate the trailing edge counter-clockwise; negative values rotate it clockwise.
Rib Thickness
Material thickness used to extrude the rib solids and to calculate rib faces and part endpoints.
Rib Count
Number of regularly positioned ribs in the panel, including its root and tip ribs.
Rib Spacing
Read-only center-to-center spacing calculated as Panel Span divided by Rib Count minus one.

Spars tab

Top Spar
Creates a rectangular wood spar on the upper airfoil surface at 25% chord. Height is vertical size; Width is chordwise size. Matching rib notches are cut.
Bottom Spar
Creates the corresponding rectangular wood spar on the lower surface at 25% chord and cuts its rib notches.
Shear Webs
Available when both top and bottom spars are selected. Creates laser-cut web parts between ribs and between the spar centers. Thickness is the sheet material thickness. Webs end at rib faces and are omitted where wood joiners occupy the bays.
CF Tube
Creates a carbon-fiber tube spar centered at 25% airfoil chord. OD cuts the rib hole and defines the outside; ID defines the bore.
CF Rod
Creates a solid carbon-fiber rod spar centered at 25% airfoil chord. OD defines both the rod and rib hole.

Rectangular top/bottom spars and the CF spar choices represent alternate 25%-chord spar systems. Choosing one system clears the other.

Sheeting

LE Top / Bottom Sheeting
Recesses the appropriate rib surface by the specified thickness and creates sheeting from the leading edge to the 25% spar. With a CF spar, sheeting ends at the spar center.
TE Top / Bottom Sheeting
Recesses the ribs and creates sheeting from the 25% spar to the trailing edge. With a CF spar, it starts at the spar center.
Stop Rib Number
The final rib covered by that sheet. The sheet begins at Rib 1 and extends to the outer face of the stop rib.

Other longitudinal members

Turbulators
Creates the selected number of square-section strips evenly spaced between the leading edge and the 25% spar. Height and Width define each strip. If LE sheeting is also selected, it is cut to fit tightly around the turbulators.
Top 60% Rear Spar
Creates a rectangular upper spar at 60% airfoil chord and cuts matching rib notches.
Bottom 60% Rear Spar
Creates a rectangular lower spar at 60% airfoil chord and cuts matching rib notches.

LE/TE tab

Leading edge

Block LE Stock
Makes a vertical cut near the leading edge, removes the rib noses, and creates a straight wood leading-edge member matching the removed region. Width is its chordwise depth; Height must accommodate the rib cut-edge height.
CF Tube LE
Cuts circular holes through the ribs and creates a uniform carbon tube for the full panel. OD defines the outside and hole diameter; ID defines the bore.
CF Rod LE
Cuts circular holes and creates a solid carbon rod. OD defines the rod and hole diameter.

Trailing edge

Sheet TE Stock
Makes a vertical rear cut and creates straight sheet trailing-edge stock matching the removed rib region. Width is chordwise depth and Height is stock height.
Slotted for Ribs
Adds rib slots to the sheet TE stock and extends the ribs into them. Slots are nominally 1/4 inch or 6 mm deep according to the design units. Slot outlines remain open where required for correct laser cutting.

LE and TE parts extend to the outer faces of the root and tip ribs. Update View stops with an error if the selected stock height cannot accommodate the corresponding rib cut edge.

Ailerons/Flaps tab

Selecting Ailerons or Flaps reveals its parameters and cuts the required control-surface region from the ribs and trailing-edge system.

Width
Chordwise depth of the control surface measured forward from the trailing edge.
Height
Vertical stock/body height used to form the control surface.
Start Rib / Stop Rib
Ribs bounding the control surface. The first rib gap in every panel remains a normal trailing-edge bay. Flaps must stop before the panel tip; ailerons may stop at the tip rib.
Hinge Post Width / Height
Cross-section of the fixed hinge post ahead of the control surface.
  • A clearance of 1/16 inch or 1.5 mm is maintained around the movable control surface.
  • If an aileron reaches the tip, its body and hinge post extend through the tip rib and that rib is cut to fit.
  • When both controls are used, Aileron Start Rib cannot be less than Flap Stop Rib.
  • If Flap Stop Rib equals Aileron Start Rib, flap and aileron widths must match. DesignRC corrects the last-edited width and uses one continuous hinge post across the shared boundary.
  • TE members remain in the unaffected first/last bays and terminate at the correct faces of ribs adjacent to controls.

Joiner tab

Add Rib 1a
Panel 1 only. Inserts an additional rib halfway between R1 and the next regular rib, useful for center-section support and joiner layout.
Center Spar Wood Joiner
Creates a laser-cut wood joiner tied to selected top/bottom 25% spars. Wood joiners occupy two rib bays and stop at rib faces rather than passing through the outer ribs. Joint ribs are split around the slot. Misaligned wood spars cause Update View to stop because a wood joiner cannot bend.
Joiner behind mid spar
Adds a CF rod, CF tube, or aluminum tube joiner at approximately 30% chord. OD controls the member/hole size; tube choices also use ID.
Joiner at 60% Chord
Adds the selected tubular/rod joiner at 60% chord with OD and, for tubes, ID.

CF and aluminum joiners pass through four ribs around a panel joint. Wood joiners pass through the two joint ribs and occupy two bays without passing through the next ribs. Joiner axes and holes are oriented perpendicular to the joint ribs so adjoining panels assemble at the intended dihedral.

Buttons and status information

Update View
Validates all panels and rebuilds ribs, cutouts, structural members, joiners, mirrored geometry, and display meshes. The cursor changes to busy while lengthy work is active.
Cancel
Appears in the status bar during Update View. Requests cancellation after the current geometry operation completes safely.
Progress bar
Shows the current rebuild percentage while status text describes the active stage.
Fit View
Fits the entire model or plan into the active graphics tab.
Generate Plan
Enabled only after valid geometry is generated. Builds the flattened, annotated full-scale drawing and switches to Plan View.
Export Plan PDF
Enabled after a plan is generated. Writes the displayed plan as a single custom-size vector PDF at 1:1 scale.
Export DXF/SVG
Opens the part-selection dialog for laser-cut part export.

The metrics area reports full wingspan, full wing area, aspect ratio, and taper ratio after a successful update.

Left mouse drag
Orbits the camera around the model.
Right mouse drag
Pans the view.
Mouse wheel
Zooms in and out.
Middle mouse button
No assigned action.
Axis gizmo
Shows the X, Y, and Z directions with their arrows meeting at a common origin.

Wood components use the wood display color, carbon-fiber components are black, and aluminum components are silver. Lighting and per-part edge contrast make adjoining pieces distinguishable.

Plan View and PDF export

The plan shows both flattened wing halves, one above the other, without dihedral foreshortening. It includes rib widths and names, LE/TE and spar outlines, sheeting, shear webs, joiners, hinge posts, control surfaces, dimensions, dihedral and twist annotations, panel spans between the halves, the R1 cutaway, border, wing metrics, and title block.

Left mouse drag
Pans the plan.
Mouse wheel
Zooms the plan.
Export Plan PDF
Starts in the last folder used for file I/O and proposes project-name.pdf. The PDF page has the plan's actual physical dimensions and a scale of 1:1.
Changing any design parameter invalidates the current plan. Run Update View and Generate Plan again before exporting a revised PDF.

DXF and SVG part export

The export dialog lists available ribs, shear webs, sheet TE parts, wood joiners, and one dihedral-angle template per panel.

DXF / SVG
Select either format or both. At least one format must be checked.
Select All
Checks every listed part.
Select None
Clears every listed part so a smaller set can be chosen.
Part checkboxes
Choose the individual files to create. Names match drawing part names, for example R1.dxf, SW1.svg, or J2.dxf.

DXF and SVG are dimensioned in millimeters for consistent CAD and laser software import. SVG files declare their physical size in millimeters. Ribs include all generated holes, notches, slots, and split-piece outlines. Centered part names are positioned to avoid cut lines where possible.

Validation and corrections

Update View validates parameter combinations before expensive geometry work. Typical checks include valid control-surface rib ranges, equal widths at a shared flap/aileron rib, required joiner material selection, straight wood-spar alignment, feasible LE/TE stock height, capped rib solids, and valid boolean cuts.

Some related fields are corrected automatically when there is one unambiguous valid value; a message explains the correction. Other errors stop processing so the indicated panel can be corrected. No previous valid preview is replaced by failed geometry.
